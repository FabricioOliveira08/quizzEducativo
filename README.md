LoopMaster – Quiz Educacional sobre Laços de Repetição

Sobre o Projeto

O LoopMaster é um quiz educacional multimídia desenvolvido em Java com JavaFX, voltado para o ensino de laços de repetição (for, while) na Educação Básica.
O sistema apresenta perguntas de múltipla escolha, com feedback visual e sonoro para respostas corretas e incorretas, incentivando o aprendizado de forma interativa.

Tecnologias Utilizadas:
- Java 21
- JavaFX
- FXML
- CSS para estilização
- Arquitetura baseada em separação por camadas (MVC simplificado)

Funcionalidades:
- Perguntas de múltipla escolha
- Modo aleatório de perguntas
- Sistema de pontuação
- Feedback visual de acerto/erro
- Sons para:
      Acerto;
      Erro;
      Vitória;
      Derrota.
- Tela de resultado final

Possíveis Melhorias Futuras:
- Banco de dados para armazenar desempenho
- Ranking de jogadores
- Novos níveis de dificuldade
- Versão web
